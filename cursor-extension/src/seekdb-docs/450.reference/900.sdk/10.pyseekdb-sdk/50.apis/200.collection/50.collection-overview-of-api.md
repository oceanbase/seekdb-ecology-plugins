# Collection management

In pyseekdb, a collection is a set of data, similar to a table in a database. You can create, query, fork, and delete collections.

The following APIs are available for collection operations:

| API | Description | Doc |
|---|---|---|
| `create_collection()` | Create a collection. | [Doc](100.create-collection-of-api.md) |
| `get_collection()` | Get a specified collection. | [Doc](200.get-collection-of-api.md) |
| `get_or_create_collection()` | Create or get a collection. If it does not exist, it is created; otherwise the existing collection is returned. | [Doc](250.get-or-create-collection-of-api.md) |
| `fork()` | Copy a collection to a new collection with the same data. | [Doc](260.fork-collection-of-api.md) |
| `list_collections()` | List all collections in the database. | [Doc](300.list-collection-of-api.md) |
| `count_collection()` | Count collections in the database. | [Doc](350.count-collection-of-api.md) |
| `has_collection()` | Check whether a collection exists. | [Doc](360.has-collection-of-api.md) |
| `delete_collection()` | Delete a specified collection. | [Doc](400.delete-collection-of-api.md) |
