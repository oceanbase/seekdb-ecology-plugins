# Database management

A database contains tables, indexes, and metadata information about database objects. You can create, query, and delete databases as needed.

For operations on a collection, you can use the following APIs.

| API | Description | Related document link |
| --- | --- | --- |
| `createDatabase()` | Creates a database. | [Documentation](200.create-database-of-seekdb-js.md) |
| `getDatabase()` | Gets a specified database. |[Documentation](300.get-database-of-seekdb-js.md)|
| `listDatabases()`  | Gets the list of databases in the instance.  |[Documentation](400.list-database-of-seekdb-js.md)|
| `deleteDatabase()`  | Deletes a specified database.|[Documentation](500.delete-database-of-seekdb-js.md)|
