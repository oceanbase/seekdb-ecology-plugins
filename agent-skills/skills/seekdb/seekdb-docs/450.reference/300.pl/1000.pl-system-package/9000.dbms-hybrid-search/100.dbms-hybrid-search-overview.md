# Overview of DBMS_HYBRID_SEARCH

The `DBMS_HYBRID_SEARCH` package is used to manage hybrid vector indexes.

## Overview of DBMS_HYBRID_SEARCH subprograms

The following table lists the `DBMS_HYBRID_SEARCH` subprograms supported by seekdb and a brief description of each.

| **Subprogram** | **Description** |
| --- | --- |
| [SEARCH](200.search.md) | Returns search results in JSON format. The results are ordered by relevance. |
| [GET_SQL](300.get_sql.md) | Returns the SQL statement that was actually executed. |
