# DQL operations

DQL (Data Query Language) operations allow you to retrieve data from collections using various query methods.

The following APIs are supported for DQL operations.

| API | Description | Related document link |
|---|---|---|
| `query()` | Vector similarity search method. | [Documentation](200.query-interfaces-of-seekdb-js.md) |
| `get()` | Query specific data by ID, document, or metadata (non-vector). |[Documentation](300.get-interfaces-of-seekdb-js.md)|
| `hybrid_search()`  | Full-text search and vector similarity search, using ranking.  |[Documentation](400.hybrid-search-of-seekdb-js.md)|
