# API reference

You can use the API to programmatically access seekdb.

## API

The following APIs are supported.

### Database

:::info
You can use the following APIs only when you connect to the database by using the `SeekdbAdminClient`. For more information about the `SeekdbAdminClient`, see [SeekdbAdminClient](100.seekdbadminclient-of-seekdb-js.md).
:::

| API | Description | Related document link |
|---|---|---|
| `createDatabase()` | Creates a database. | [Documentation](110.database-of-seekdb-js/200.create-database-of-seekdb-js.md) |
| `getDatabase()` | Gets a specified database. |[Documentation](110.database-of-seekdb-js/300.get-database-of-seekdb-js.md)|
| `listDatabases()`  | Gets the list of databases in the instance.  |[Documentation](110.database-of-seekdb-js/400.list-database-of-seekdb-js.md)|
| `deleteDatabase()`  | Deletes a specified database.|[Documentation](110.database-of-seekdb-js/500.delete-database-of-seekdb-js.md)|

### Collection

:::info
You can use the following APIs only when you connect to the database by using the `SeekdbClient`. For more information about the `SeekdbClient`, see [SeekdbClient](50.seekdbclient-of-seekdb-js.md).
:::

| API | Description | Related document link |
|---|---|---|
| `createCollection()` | Creates a collection. | [Documentation](200.collection-of-seekdb-js/100.create-collection-of-seekdb-js.md) |
| `getCollection()` | Gets a specified collection. |[Documentation](200.collection-of-seekdb-js/200.get-collection-of-seekdb-js.md)|
| `getOrCreateCollection()` | Creates or queries a collection. If the collection does not exist in the database, it is created. If it exists, the corresponding result is obtained. |[Documentation](200.collection-of-seekdb-js/250.get-or-create-collection-of-seekdb-js.md)|
| `fork()` | Forks a specified collection to create a new collection with the same data.|[Documentation](200.collection-of-seekdb-js/260.fork-collection-of-seekdb-js.md)|
| `listCollections()`  | Gets the list of collections in the database.  |[Documentation](200.collection-of-seekdb-js/300.list-collection-of-seekdb-js.md)|
| `countCollection()`  | Counts the number of collections in the database.  |[Documentation](200.collection-of-seekdb-js/350.count-collection-of-seekdb-js.md)|
| `hasCollection()`  | Checks whether a specified collection exists in the database.  |[Documentation](200.collection-of-seekdb-js/360.hascollection-collection-of-seekdb-js.md)|
| `deleteCollection()`  | Deletes a specified collection.|[Documentation](200.collection-of-seekdb-js/400.delete-collection-of-seekdb-js.md)|

### DML

:::info
You can use the following APIs only when you connect to seekdb by using the SeekdbClient. For more information about the SeekdbClient, see [SeekdbClient](50.seekdbclient-of-seekdb-js.md).
:::

| API | Description | Related document link |
|---|---|---|
| `add()` | Inserts a new record into the collection. | [Documentation](300.dml-of-seekdb-js/200.add-data-of-seekdb-js.md) |
| `update()` | Updates an existing record in the collection. |[Documentation](300.dml-of-seekdb-js/300.update-data-of-seekdb-js.md)|
| `upsert()`  | Inserts a new record or updates an existing record.  |[Documentation](300.dml-of-seekdb-js/400.upsert-data-of-seekdb-js.md)|
| `delete()`  | Deletes a record from the collection.|[Documentation](300.dml-of-seekdb-js/500.delete-data-of-seekdb-js.md)|

### DQL

:::info
You can use the following APIs only when you connect to seekdb by using the SeekdbClient. For more information about the SeekdbClient, see [SeekdbClient](50.seekdbclient-of-seekdb-js.md).
:::

| API | Description | Related document link |
|---|---|---|
| `query()` | A vector similarity search method. | [Documentation](400.dql-of-seekdb-js/200.query-interfaces-of-seekdb-js.md) |
| `get()` | Queries specific data from the table by using the ID, document, and metadata (non-vector). |[Documentation](400.dql-of-seekdb-js/300.get-interfaces-of-seekdb-js.md)|
| `hybridSearch()`  | A hybrid search method that combines full-text search and vector similarity search.  |[Documentation](400.dql-of-seekdb-js/400.hybrid-search-of-seekdb-js.md)|
