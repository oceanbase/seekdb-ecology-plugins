# Collection management

In seekdb-js, a collection is a set of data, similar to a table in a database. You can create, query, and delete collections.

The following APIs are available for managing collections.

| API | Description | Related document link |
|---|---|---|
| `createCollection()` | Creates a collection. | [Documentation](100.create-collection-of-seekdb-js.md) |
| `getCollection()` | Retrieves a specified collection. |[Documentation](200.get-collection-of-seekdb-js.md)|
| `getOrCreateCollection()` | Creates or retrieves a collection. If the collection does not exist in the database, it is created; if it exists, the corresponding result is retrieved. |[Documentation](250.get-or-create-collection-of-seekdb-js.md)|
| `fork()` | Copies a specified collection to create a new collection with the same data.|[Documentation](260.fork-collection-of-seekdb-js.md)|
| `listCollections()`  | Retrieves a list of collections in the database.  |[Documentation](300.list-collection-of-seekdb-js.md)|
| `countCollection()`  | Counts the number of collections in the database.  |[Documentation](350.count-collection-of-seekdb-js.md)|
| `hasCollection()`  | Checks whether a specified collection exists in the database.  |[Documentation](360.hascollection-collection-of-seekdb-js.md)|
| `deleteCollection()`  | Deletes a specified collection.|[Documentation](400.delete-collection-of-seekdb-js.md)|
