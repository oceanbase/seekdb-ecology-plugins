# DML operations

DML (Data Manipulation Language) operations allow you to insert, update, and delete data in a collection.

The following APIs are supported for DML operations.

| API | Description | Related document link |
|---|---|---|
| `add()` | Inserts a new record into the collection. | [Documentation](200.add-data-of-seekdb-js.md) |
| `update()` | Updates an existing record in the collection. |[Documentation](300.update-data-of-seekdb-js.md)|
| `upsert()`  | Inserts a new record or updates an existing record.  |[Documentation](400.upsert-data-of-seekdb-js.md)|
| `delete()`  | Deletes a record from the collection.|[Documentation](500.delete-data-of-seekdb-js.md)|
